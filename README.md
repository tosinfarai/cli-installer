# sca-cloud-school-application
## Prerequisites
### Python version
- python 3
### For Linux
- Ubuntu flavor
### For Windows
- [Scoop](https://scoop.sh/) package manager
### For Mac
- [Homebrew](https://brew.sh/) package manager

## Usage
To install Node.js:
- `python3 mycli.py --install nodejs`

To install wget:
- `python3 mycli.py --install wget`

To install curl:
- `python3 mycli.py --install curl`
